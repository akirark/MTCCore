MTCCore
=============

# Introduction
MTCCore is a core of the MultiTextConverter.
It can detect line breaks and some text encodings.

Supported text encodings are:
* UTF-8
* UTF-16
* UTF-32
* Shift-JIS
* JIS (ISO-2022-JP)
* EUC-JP

Supported line breaks are:
* CR
* LF
* CR + LF
* LS
